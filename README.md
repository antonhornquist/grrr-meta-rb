# grrr-meta-rb

Class definitions and utility methods for generating class file stubs for [Grrr-sc](http://github.com/antonhornquist/Grrr-sc) and [grrr-rb](http://github.com/antonhornquist/grrr-rb).

## Requirements

This library requires [rsclass-rb](http://github.com/antonhornquist/rsclass-rb).

A number of environment variables also need to be set for editing and generation to work. See grrr_meta.rb source for details.
